-----------------------------------------------------------------------------------------------------------------------------------------------------------------

RESULT     :  Displayed in Function Body
Example Program :  ApproachEx2.py
-----------------------------------------------------------------------------------------------------------------------------------------------------------------

-----------------------------------------------------------------------------------------------------------------------------------------------------------------

RESULT     :  Displayed in Function Body

Example Program :     ApproachEx3.py
